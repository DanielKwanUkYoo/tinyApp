# TinyApp Project

TinyApp is an application built with Node and Express to shorten any of longURLs for user's convenience such as sharing link on a chatApp.

## Dependicies

- Express
- Node.js
- EJS
- Cookie-Parser
- Body-Parser
- Bcrypt

## How to Start

- Install all dependicies
- Run by "npm start" on your commandline
